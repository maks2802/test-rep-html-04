Основні поняття: URL

Для встановлення розташування Web-сторінок у World Wide Web використовується URL (Uniform Resource Locator).

URL був винайдений Тімом Бернерсом-Лі у 1990 році у стінах Європейської ради з ядерних досліджень у Женеві, Швейцарія. URL стала фундаментальною інновацією в Інтернеті. Спочатку URL призначався для позначення місць розташування ресурсів (найчастіше файлів) у Всесвітньому павутинні. Сьогодні URL використовується для позначення адрес майже всіх ресурсів Інтернету. Стандарт URL закріплено в документі RFC 1738, колишня версія була визначена в RFC 1630. Зараз URL позиціонується як частина більш загальної системи ідентифікації ресурсів URI, термін URL поступово поступається місцем ширшому терміну URI. Стандарт URL регулюється організацією IETF та її підрозділами.

Приклади URL:

  • https://www.google.com/- http-сервер повинен знайти та надіслати клієнту (браузеру) стартову сторінку google;

  • https://www.onmu.odessa.ua/- http-сервер повинен знайти та відправити клієнту (браузеру) стартову сторінку ОНМУ.

URL включає:

  1. Метод доступу до ресурсу, тобто протокол доступу (http-доступ до Web-сторінки у мережі з використанням відповідного протоколу). Це найчастіше використовуваний метод доступу до будь-якого HTML-документа у мережі.
  2. Мережева адреса ресурсу.
  3. Повний шлях до файлу на сервері.
